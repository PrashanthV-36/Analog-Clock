# Simple Analog Clock

A lightweight web project that displays a real-time analog clock using **HTML, CSS, and JavaScript**. The clock updates dynamically with smooth second, minute, and hour hand movements, mimicking a traditional wall clock.

---

## 🚀 Features
- Real-time analog clock with smooth ticking hands
- Built with pure **HTML, CSS, and JavaScript** (no external libraries)
- Minimal, clean, and responsive design
- Easy to integrate into any webpage

---

## 🛠️ Tech Stack
- **HTML5** – Structure
- **CSS3** – Styling
- **JavaScript (ES6+)** – Logic

---

## ⚡ How to Use Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/simple-analog-clock.git
   ```

2. Navigate into the project folder:
   ```bash
   cd simple-analog-clock
   ```

3. Open **index.html** in your browser.
